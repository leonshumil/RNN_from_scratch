# RNN from Scratch in C

A minimalist aproch for creating RNN from scratch using only basic C libraries, this project is a spiritual continuetion of a project i did in uni where i implamented Neural  Network using MATLAN from scratch back in 2022. 

---

## 🚧 Status: In Development

This project is currently under active development. The goal is to implement a basic RNN framework, including forward propagation, backpropagation through time (BPTT), and training on simple sequential datasets.

---

## 📚 References & Resources

The theoretical background, equations, and logic for this implementation are heavily inspired by and referenced from the following resources:

* **IBM Think: "What are Recurrent Neural Networks?"** - [Link to Article](https://www.ibm.com/think/topics/recurrent-neural-networks#763338458) - A great high-level overview of RNN architectures and applications.
* **Towards Data Science: "The Math Behind Recurrent Neural Networks"** by Simeon Kostadinov - [Link to Article](https://towardsdatascience.com/the-math-behind-recurrent-neural-networks-2de4e0098ab8/) - A critical deep dive into the mathematical derivations and BPTT, which is the direct inspiration for the math implemented in this repository.
